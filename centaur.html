Hiding inside a DGT Centaur Chess Board is a Raspberry Pi Zero running stockfish and some custom code to interface with the board.

This opens up the door to running other chess engines and redesigning the menu and screens that the board displays on its e-paper screen.

But that's not all. Replace the board with a Raspberry Pi Zero 2W and you've got a faster chess board with Wifi support. I started hacking it and was soon joined by others. Check out the project for alternative DGT Centaur software at DGTCentaurMods
