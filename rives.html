One day whilst browsing the itch.io game jams pages, I came across one for rives.io. RIVES is an on-chain fantasy console.

Because of the setup and the minimalist API this very much has the feel of coding for a retro system. Which is what intrigued me.

I entered two games. A first game where I was mostly learning the system (called Blockie) and a second game based on my fondness for Shooters (called Peeyow)

I will embed both games on this page. But it's advisable to either finish one or use the stop button before playing the next (so you don't have two games running taking computer resources.

First blockie. You know roughly what to do :) Use the left and right arrow keys to shift the blocks. Use up to rotate the colors. Use down to drop the block now. Try to match up 4 or more colors that connect on either horizontal or vertical axis (in other words, no diagonals)

Next up Peeyow. Use the arrow keys and F for fire. Shoot the baddies, collect powerups, avoid lasers. Simples :)
